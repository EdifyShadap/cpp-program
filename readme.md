Simple Calculator in C++

This is a basic console-based calculator written in C++ using Object-Oriented Programming (OOP).  
It supports the following operations:

- Addition  
- Subtraction  
- Multiplication  
- Division  

The program repeatedly asks the user for an operation until they choose to exit.

---

Features
- Implemented using a `calculator` class  
- Menu-driven interface  
- Handles division-by-zero error  
- Easy to modify and extend  
- Beginner-friendly structure

---

How It Works
The class `calculator` contains:
- Two float variables (`a`, `b`)  
- A menu function `option()`  
- Four member functions for operations:
  - `addition()`
  - `subtraction()`
  - `multiplication()`
  - `division()`

The program asks the user for input and performs the selected operation.

---

 How to Run the Program

Compile:
g++ calculator.cpp -o calculator
